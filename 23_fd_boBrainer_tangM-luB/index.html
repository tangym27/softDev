    Setup

    The fastest way to start using the Foundation features showcased on this site is to simply add the CDN link into your HTML head section.

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.0/dist/css/foundation.min.css">
    Alternatively, download the complete version here for all the features available in the framework.
